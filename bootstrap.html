BOOTSTRAP Grid

BOOTSTRAP uses a 12-colunm grid system for responsive layout

Example: two column 

                    
                        <div class="container">
                            <div class="row">
                                <div class="col-md-6">column 1</div>
                                <div class="col-md-6"column 2</div>
                            </div>
                        </div>

components: navbar

bootstrap provide pre-built components like navbar

                        
                       <nav class="navbar navbar-expand-sm bg-light">
                            
                <ul class="navbar-nav">
                           <li class="nav-item">
                           <a class="nav-link" href="#">Link 1</a>

Utilities:Float and clear fix

Float an element to the right with the .float-right class or to the left with .float-left, and clear floats with the .clearfix class:

                        
                            <div class="clearfix">
                                  <span class="float-left">Float left</span>
                                    <span class="float-right">Float right</span>
                            </div>
