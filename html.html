HTML Basics

HTML stands for HyperText Markup Language

                                
                                <!DOCTYPE html>
                                <html lang="en">
                                <head>
                                    <meta charset="UTF-8">
                                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                    <title>Document </title>
                                </head>
                                <body>
                                    <h1>hello</h1>
                                    <h2>world</h2> 
                                </body>
                                </html>

HTML Tables

HTML Tables are used to create complex layouts on a webpage

                                
                                <!DOCTYPE html>
                                <html lang="en">
                                <head>
                                    <meta charset="UTF-8">
                                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                    <title>Document </title>
                                </head>
                                <body>
                                    <table>
                                        <tr>
                                            <td></td>
                                        </tr>
                                    </table>
                                </body>
                                </html>

HTML Form

HTML Form is used to collect information from a user

                                
                                <!DOCTYPE html>
                                <html lang="en">
                                <head>
                                    <meta charset="UTF-8">
                                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                    <title>Document </title>
                                </head>
                                <body>
                                    <form action="">
                                        <input type="text" name="fullname" id="fullname">
                                        <input type="number" name="age" id="age">
                                    </form>
                                </body>
                                </html>
